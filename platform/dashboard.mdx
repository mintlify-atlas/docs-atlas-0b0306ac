The Mantlz dashboard is your central hub for managing forms, tracking submissions, and monitoring campaign performance. Designed for clarity and efficiency, the dashboard provides quick access to all platform features.

## Dashboard Home

When you log in to Mantlz, you're greeted with a personalized dashboard that displays:

<CardGroup cols={2}>
  <Card title="Forms Overview" icon="file-lines">
    View all your forms at a glance with submission counts and creation dates
  </Card>
  <Card title="Quick Stats" icon="chart-simple">
    See total responses collected across all your forms
  </Card>
  <Card title="Recent Activity" icon="clock">
    Monitor the latest form submissions and campaign activities
  </Card>
  <Card title="Quick Actions" icon="bolt">
    Create new forms or campaigns directly from the dashboard
  </Card>
</CardGroup>

## Navigation

The dashboard provides seamless navigation to all platform sections:

### Main Navigation Menu

- **Forms** - Access your form builder and manage existing forms
- **Logs** - View detailed submission history and search through responses
- **Campaigns** - Create and manage email campaigns
- **Billing** - Manage subscriptions and view invoices

### Welcome Section

The welcome banner displays:

- Your profile avatar and name
- Total form count
- Personalized greeting based on your account activity

## Forms List View

Your forms are displayed in an organized grid layout showing:

- **Form name and type** (Waitlist, Contact, Feedback, etc.)
- **Creation date** - When the form was created
- **Response count** - Total submissions received
- **Email settings status** - Whether auto-responses are enabled

### View Modes

Switch between two viewing options:

<Tabs>
  <Tab title="Grid View">
    Cards display forms with visual emphasis on key metrics
  </Tab>
  <Tab title="List View">
    Compact table format for efficient scanning of multiple forms
  </Tab>
</Tabs>

<Tip>
Use the view mode toggle in the top-right corner to switch between grid and list views. Your preference is saved in the URL for easy sharing.
</Tip>

## Pagination

When you have multiple forms:

- Navigate through pages using arrow buttons
- Default display shows 8 forms per page
- Page numbers indicate your current position
- Total form count displayed in the footer

## Empty State

If you haven't created any forms yet, the dashboard shows:

- A welcoming message encouraging you to create your first form
- Direct access to the form builder
- Quick links to documentation and templates

## Quick Actions

### Create New Form

Click the "Create Form" button to launch the form builder with options for:

- Waitlist forms
- Contact forms
- Feedback forms
- Survey forms
- Application forms
- RSVP forms
- Custom forms
- Analytics opt-in forms

### View Form Details

Click any form card to access:

- Detailed analytics
- Submission history
- Form settings
- Export options

## Payment Success Modal

After completing a subscription purchase, a success modal automatically appears:

- Confirms your payment
- Shows your new plan features
- Provides next steps for getting started

<Info>
The dashboard uses real-time data queries to ensure you always see the most up-to-date information about your forms and submissions.
</Info>

## Responsive Design

The dashboard adapts to different screen sizes:

- **Desktop** - Full feature set with optimized layouts
- **Tablet** - Adjusted spacing and collapsible elements
- **Mobile** - Stacked cards and simplified navigation

## Loading States

While data loads, you'll see skeleton screens that:

- Maintain layout structure
- Indicate loading progress
- Provide visual feedback during data fetching

<Note>
All dashboard queries include automatic retry logic and error handling to ensure reliability even with poor network connections.
</Note>
